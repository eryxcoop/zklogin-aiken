# zkLogin in Aiken
Open source implementation of the zkLogin protocol in Aiken validators and circom circuits.

The report can be found under the ```report/``` directory.

The video for the Milestone 1 can be found in the ```milestone_1.mp4``` file.

## Dependencies
* ```Aiken (1.1.19)```

## Run the tests
1) Go to the ```src-backend/``` directory
2) Run ```aiken-check```

## Building
1) Go to the ```src-backend/``` directory
2) Run ```aiken-build```
